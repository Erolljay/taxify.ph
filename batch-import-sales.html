📤 Batch Import — Sales Invoices
Fill a simple Excel sheet, then post Sales Invoices (and same-day Receipts) directly to Manager
ℹ️ This tool posts directly to Manager via its API — no manual upload needed. Each row's Customer name is matched against existing contacts; if it looks like a possible duplicate, the preview lets you pick the matching contact (or confirm it's new) right there. The Sales Invoice is then created, then (for rows marked "Paid Same Day") a matching Receipt that settles it. Review the row-by-row results after posting — if a row fails, fix the source data and click "Post to Manager" again to retry just the failed rows.
