📤 Batch Import — Payroll
Fill a simple Excel sheet, then post Payslips (and same-day net-pay Payments) directly to Manager
ℹ️ This tool posts directly to Manager via its API — no manual upload needed. For each row: a missing Employee is auto-created (name only), then a Payslip is created using whichever payslip item the firm has already assigned (under "Payslip items") to each BIR earnings/deduction/contribution category, then a Payment is created for the net pay (all payroll is assumed paid the same day as the pay period end date). Review the row-by-row results after posting — if a row fails, fix the source data and click "Post to Manager" again to retry just the failed rows.
